# dir
A local GitHub repository created with Claude Code

## 项目简介

这是一个使用 Claude Code 创建的示例 GitHub 仓库，展示了完整的项目结构和代码示例。

## 项目结构

```
dir/
├── src/           # 源代码目录
│   ├── main.cpp   # C++ 示例程序
│   └── utils.py   # Python 工具模块
├── docs/          # 文档目录
│   └── README.md  # 详细文档
├── tests/         # 测试目录
│   └── test_utils.py  # 单元测试
└── .gitignore     # Git 忽略文件
```

## 快速开始

### 运行 C++ 程序
```bash
cd src
g++ main.cpp -o main
./main
```

### 运行 Python 程序
```bash
python3 src/utils.py
```

### 运行测试
```bash
python3 tests/test_utils.py
```
