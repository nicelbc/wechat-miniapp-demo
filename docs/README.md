## 快速开始

### 环境要求

- C++ 编译器 (g++/clang++)
- Python 3.6+

### 运行示例

```bash
# 编译并运行 C++ 程序
g++ src/main.cpp -o main
./main

# 运行 Python 程序
python3 src/utils.py
```
